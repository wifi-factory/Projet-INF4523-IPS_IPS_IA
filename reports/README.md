# Rapports et supports

Le dossier `reports/` regroupe les sorties documentaires conservees avec le
depot.

## Sous-dossiers principaux

- `model_training/` : evaluation du modele `random_forest_lab_v2`
- `week2_preprocessing/` : documentation et figures du pretraitement
- `week3_modelisation/` : rapport intermediaire de modelisation et assets
- `dashboard_mockups/` : maquettes de dashboard
- `live_tests/` : petites preuves de validation live conservees en JSON

## Ce qui n'est pas une source de verite documentaire

Certains logs de lancement backend/dashboard sont generes localement dans ce
dossier pendant les tests. Ils sont ignores par Git et ne font pas partie des
livrables a remettre.
